### Get Pod name

````shell

kubectl -it exec $POD -- bash
````

### Get log

````shell
kubectl logs $POD

kubectl get cm

kubectl exec $POD -- ls /etc/logging.properties

kubectl exec $POD -- cat /etc/logging.properties

kubectl exec $POD -- cat /opt/bonita/BonitaCommunity-2021.1/server/conf/logging.properties
````
